Authentication flow
POST
/api/auth/register → bcrypt.hash(password) → prisma.user.create
POST
/api/auth/callback/credentials → bcrypt.compare → JWT session { id, name, email, role }
Guard
All protected pages call getServerSession(authOptions) → redirects to /login if null
Quick start
# 1 install
npm install
# 2 generate + migrate
npx prisma generate && npx prisma migrate dev
# 3 run
npm run dev → http://localhost:3000
# .env required
DATABASE_URL="file:./dev.db"
NEXTAUTH_SECRET="your-secret-here"
NEXTAUTH_URL="http://localhost:3000"
OPENWEATHER_API_KEY="optional"
